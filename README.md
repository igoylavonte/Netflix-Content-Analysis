# 🎬 Netflix Content Analysis

## 📌 Project Overview
This project analyzes Netflix content using Power BI to identify trends in genres, ratings, and release years.

## 🛠️ Tools Used
- Power BI
- Microsoft Excel

## 📈 Key Insights
- Drama dan Comedy termasuk genre dengan jumlah konten terbanyak.
- Jumlah rilis konten meningkat pada tahun-tahun terbaru.
- Distribusi rating menunjukkan mayoritas film berada pada kisaran rating tinggi.

# Netflix-Content-Analysis

Netflix Content Analysis Dashboard adalah dashboard interaktif yang dibuat menggunakan Microsoft Power BI untuk menganalisis data film dan serial netflix. Dashboard ini menampilkan berbagai informasi penting seperti pertumbuhan konten berdasarkan tahun rilis, distribusi rating film. genre yang paling banyak muncul. dan rating tertinggi

## 🚀 Features
- Netflix Content Growth
- Top 10 Genre
- Films by Rating
- Top Movies by Rating
- Interactive Filters (Genre, Release Year, Title)
